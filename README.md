## Install

```sh
npm i
```

## Run

You have to start the server first
```sh
npm run server
```

and then start the client
```sh
npm run dev

# or

npm run build
npm run preview
```


## Run with Docker

- Run client `bash run.sh`, see [run.sh](run.sh)
- And then run server [vue-chat-server](https://github.com/krchmkn/vue-chat-server/tree/main)
